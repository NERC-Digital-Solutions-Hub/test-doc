Remarks

Use the DateTime type whenever date with time values are required in a POCO. If the DateTime property must be stored in its own table (for example, to satisfy advanced geospatial metadata requirements) map it to this DateTimeDbEntity. This design helps ensure consistent handling of DateTime values according to the ISO 19115 standard.
