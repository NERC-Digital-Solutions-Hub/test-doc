NilReason

The NilReason is a string value of one of the following three options:

  1. The value is a low case string representation of one of the values of the NilReasonEnumeration, except the value Other
  2. The value is a combination of the low case string representation of the value Other followed by : and then by a string providing a brief explanation, where text is a string of two or more characters with no included spaces.
    Example: other:brief explanation goes here
    The entire string must be valid against the following regular expression:
    other:\w{2,}
  3. The value is a Uniform Resource Identifier (URI).
    A particular community may choose to assign more detailed semantics to the standard values provided by the NilReasonEnumeration. Alternatively, the URI method enables a specific or more complete explanation for the absence of a value to be provided elsewhere and indicated by-reference in an instance document.

Example: 

NilReason = INilReason.GetNilReason(NilReasonEnumeration.Missing);
NilReason = INilReason.GetNilReason(NilReasonEnumeration.Other, "Example reason give here");
NilReason = INilReason.GetNilReason("http://www.example.com/a-detailed-explanation-for-nilreason/";

GetNilReason(NilReasonEnumeration, string)

Sets the string value of the NilReason property.

The procedure makes sure that rules for setting the string are observed and should be preferably used rather than setting the string value using the property's setter. The reason for having a public setter in the NilReason property is to allow for XML deserialization.
