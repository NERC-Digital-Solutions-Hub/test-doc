ArcRole

The xlink:arcrole attribute is used to classify or specify the function of a link in a machine-readable way. It holds a URI that provides semantic meaning to the relationship between resources.

Uses:

  • The xlink:arcroleattribute is typically used in extended links and arcs to define the nature of relationships between linked resources.
  • Used to categorize links in structured XML documents, like financial data, legal references, or scientific publications.
  • Used in GIS applications to define topological relationships (e.g., "is next to," "contains," "overlaps").
  • Used in financial reporting to define relationships between accounting concepts.

Href

The xlink:href attribute is used to define the target of a hyperlink in XML-based documents. It works similarly to the href attribute in HTML but is used in a more structured way within XML.

Uses:

  • It is used to link external resources such as web pages, images, datasets, or documents within XML-based formats like SVG, GML (Geospatial Markup Language), and XBRL (Financial Reporting).
  • Used in structured metadata to point to external datasets, images, or documents.
  • Used to reference geographic features in GIS (Geospatial Information Systems).
  • Used in financial reporting to link definitions, schemas, or taxonomies.
[JsonProperty("href")]
public string Href { get; set; }

Property Value

string

Remarks

The xlink:href is a locator attribute.

Role

The xlink:role attribute is used to describe the intended role or purpose of a linked resource. Unlike xlink:arcrole, which defines the relationship between two resources, xlink:role provides context about the linked resource itself.

Uses:

  • It is typically used in simple links, extended links, and resource definitions to describe what the linked resource represents.
  • Used to classify linked documents, images, datasets, and multimedia.
  • Used in GML (Geography Markup Language) to describe geospatial data roles.
  • Used in XBRL (eXtensible Business Reporting Language) to define financial document roles.
public string Role { get; set; }

Property Value

string

Remarks

The xlink:role is a semantic attribute.
