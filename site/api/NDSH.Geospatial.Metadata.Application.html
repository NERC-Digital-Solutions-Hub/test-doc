Namespace NDSH.Geospatial.Metadata.Application

Classes

AbstractDS_Aggregate_Type

Represents an identifiable collection of datasets in the ISO 19115 metadata standard. This abstract class serves as a base type for various dataset aggregations.

DS_DataSet_Type

Represents an identifiable collection of data. This class models a dataset that may contain multiple metadata records (Has) and may be part of one or more dataset aggregates (PartOf).

DS_Initiative_Type

Represents an initiative within a dataset aggregate in geospatial metadata. This class inherits from AbstractDS_Aggregate_Type and provides a specialized dataset aggregate under the ISO 19115 metadata standard.

DS_OtherAggregate_Type

Represents a generalized dataset aggregate in geospatial metadata. This class is used for aggregates of datasets that do not fall into predefined categories.

DS_Platform_Type

Represents a geospatial platform used as a basis for dataset series within metadata applications. This class extends DS_Series_Type, allowing it to be part of dataset collections.

DS_ProductionSeries_Type

Represents a production series in geospatial metadata,
which is a specialized dataset series related to a production process.

DS_Sensor_Type

Represents a sensor-based dataset series in geospatial metadata.

DS_Series_Type

Represents a series of datasets in geospatial metadata.

DS_StereoMate_Type

Represents a stereo mate dataset, which is a dataset that is related to another dataset through stereo imaging principles.
