Enum MinutesSecondsValueType

Specifies how the minute and second portions of a degrees-minutes-seconds (DMS) angle are stored or interpreted within a GML context. This enumeration allows for either a single decimal value for minutes or separate integer fields for minutes and seconds.

DecimalMinutes = 0

Represents the minute portion of a DMS angle as a single decimal value (e.g., 15.5 minutes).

Minutes = 1

Represents the minute portion of a DMS angle as a whole-number integer.

Seconds = 2

Represents the second portion of a DMS angle as a decimal or integer value.

Remarks

Defined in /2005/gml/measures.xsd, these values are typically used alongside the DMSAngleType to distinguish the format in which the minute and second components are provided. For example, one might specify the minutes portion as a decimal number in DecimalMinutes, or use integer Minutes with optional integer Seconds.

Note that this approach to specifying angles is marked as deprecated in some GML definitions and may only be retained for backward compatibility with older datasets or schemas.
