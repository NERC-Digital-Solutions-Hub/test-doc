GetCollection<TCollection>(XmlElementReader, string, XElement, XmlElementSerializerOptions)

Gets a ICollection of read elements from an XElement children. This is designed for reading ISO 19115 elements. Therefore, it will use GetPropertyElement(XElement) to get the first child element of the specified elements.

public static TCollection GetCollection<TCollection>(this XmlElementReader reader, string elementName, XElement element, XmlElementSerializerOptions options) where TCollection : IList, new()

Parameters

reader XmlElementReader

The XmlElementReader instance.

elementName string

The name of the elements to added to the collection.

element XElement

The element to read. This is the element which should contain the child elements with the elementName.

options XmlElementSerializerOptions

The XmlElementSerializerOptions instance.

Returns

TCollection

An instance of TCollection containing read elements with the elementName.

Type Parameters

TCollection

The collection Type.

Remarks

Since this method is designed specifically for reading ISO 19115 elements, it is added as an extension method to the XmlElementReader class, rather than in the class itself.
