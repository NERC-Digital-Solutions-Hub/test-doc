Remarks

The IObjectReference interface extends IAssociation and INullableUuidRef, integrating metadata linking mechanisms and UUID-based identification.

Key Features:

  • Supports linking via IAssociation to reference remote resources.
  • Provides a UUID reference mechanism via INullableUuidRef.
  • Ensures compliance with GML and ISO 19115 metadata standards.

Considerations:

  • The implementation derives from IAssociation for simplicity, ensuring alignment with AssociationAttributeGroup.
  • Designed for referencing objects within geospatial metadata structures.
