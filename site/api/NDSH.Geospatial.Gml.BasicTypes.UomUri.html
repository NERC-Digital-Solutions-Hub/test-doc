Remarks

A valid UOM URI must start with one of the following:

  • "#" (fragment identifier).
  • "./" or "../" (relative path).
  • A scheme name followed by ":" (e.g., "http:", "urn:", "ftp:").
This ensures compatibility with absolute and relative URIs.
