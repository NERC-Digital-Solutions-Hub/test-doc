Represents the GML AngleChoiceType as defined by GML specification. This type has been marked as deprecated (obsolete) in the underlying schema.
It provides a choice between specifying an angle as a single numeric measurement AngleType or as a degrees-minutes-seconds representation DMSAngleType.

Remarks

Per the schema definitions in /2005/gml/measures.xsd, AngleChoiceType includes two potential XML elements:

  • AngleType serialized as angle
  • DMSAngleType serialized as dmsAngle
Both AngleType and DMSAngleType are also marked deprecated. This class is therefore provided primarily for backward compatibility with legacy GML data and is not recommended for use in new designs.

The choice of angle representation is stored in the Angle property, which can contain an instance of either AngleType or DMSAngleType. Although this class inherits from ObservableModel, it is marked ObsoleteAttribute to reflect the GML schema's deprecation notice.

Remarks

In the underlying GML schema, this property corresponds to either the AngleType or the DMSAngleType. Both formats are marked deprecated in the specification but are preserved here for compatibility with legacy datasets.

Because AngleChoiceType is itself marked Obsolete, and its usage in new designs is discouraged, consider modern alternatives for representing angles if forward-compatibility and standardization are a priority.
