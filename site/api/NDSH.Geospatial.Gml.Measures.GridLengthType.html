Remarks

Defined in /2005/gml/measures.xsd, GridLengthType inherits the properties of MeasureType, thus including both a numeric Value (base type double) and a UnitOfMeasure (uom).
This is commonly used to specify the spacing along a grid axis, ensuring consistency in geospatial data where precise measurements are needed.

By leveraging GridLengthType, you can maintain explicit unit information alongside the numeric value, in accordance with GML standards for expressing distance or resolution within grid-based spatial datasets.
