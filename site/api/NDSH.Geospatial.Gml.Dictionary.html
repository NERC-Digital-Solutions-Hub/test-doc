Classes

DefinitionBaseType

Represents the base type for definitions in GML. This class serves as a restriction of AbstractGMLType, implementing the IStandardObjectProperties interface.

DefinitionProxyType

Represents a proxy definition that refers to an external definition rather than containing its own definition.

DefinitionType

The DefinitionType specifies a definition, which can be included in or referenced by a dictionary. The content model for a generic definition is a derivation from AbstractGMLType.

  • The Description property shall hold the definition if this can be captured in a simple text string.
  • The DescriptionReference property may carry a link to a description elsewhere.
  • The Identifier property shall provide one identifier identifying this definition.
  • The Identifier property shall be unique within the dictionaries using this definition.
  • The Name property shall provide zero or more terms and synonyms for which this is the definition.

The Remarks property shall be used to hold additional textual information that is not conceptually part of the definition but is useful in understanding the definition.

DictionaryEntryType

A DictionaryEntryType contains or refers to the definitions which are members of a dictionary.

DictionaryType

Sets of definitions may be collected into dictionaries or collections. A DictionaryType is a non-abstract collection of Definitions .

IndirectEntryType

Represents an indirect entry in a GML dictionary.
