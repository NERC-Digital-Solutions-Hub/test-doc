Enum NilReasonEnumeration

Namespace
NDSH.Geospatial.Gml.BasicTypes

Enumerates standardized reasons for a property or attribute to have a null (missing) value, following the GML (ISO 19136) nil-reason pattern. This provides clarity on why a value is unavailable or not applicable at a given time.
